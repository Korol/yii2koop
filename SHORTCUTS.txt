Выделить текст от курсора до начала следующего слова, если его нет – то до конца текущего слова:
Option + Shift + Left (влево) или Option + Shift + Right (вправо)
Перемещение по тексту – всё аналогично, только без Shift.

!!! Расширение выделения - экономит тонны времени и кликов по клаве при редактировании и кодинге:
Option + Up - выделяет слово под курсором
Последующие нажатия Up при зажатом Option - последовательно выделяет:
    Для простого текста:
        предложение
        абзац
        весь текст на странице
    Для кода:
        переменная
        свойство
        весь код в строке
        вся строка
        связанные фрагменты кода, причем последовательно расширяя границы связанности
Option + Down - выполняет обратное действие, т.е. последовательно снимает выделение в обратном порядке.

!!! Мульти-курсор - тоже гуд:
Зажать Option, затем установить произвольное количество курсоров в нужных местах кода.

Удаление текста справа от курсора:
Fn + Delete

Мульти-выделение вхождений слова с простановкой мульти-курсора:
Ctrl + G - зажать Ctrl, затем последовательными нажатиями G выделить нужные вхождения этого слова.
Курсоры проставляются автоматически, на ту же позицию, как и в исходном слове.
Поиск вхождений происходит сверху вниз.

Мульти выделение столбцов, с простановкой мульти-курсоров:
Переключить режим выделения текста: Edit -> Column Selection Mode или комбинация Shift + Cmd + 8
Мышкой выделить нужные столбцы.
Мульти-курсоря автоматически проставятся на место окончания выделения столбцов.

Дублировать строку Cmd + D

Удалить строку Cmd + Delete

Переместить строку: Option + Shift + Up - вверх, Option + Shift + Down - вниз

Переместить выделенный фрагмент кода: Shift + Cmd + Up - вверх, Shift + Cmd + Down - вниз

Автокомплит: Ctrl + Space

Меняем регистр: Cmd + Shift + U

Навигационная панель – это альтернатива Project View (меню проекта слева, по Cmd + 1): Cmd + Up
Работа с панелью производится при помощи стрелок и Enter.

Структура документа - это альтернатива вкладке Structure (слева, по Cmd + 7): Cmd + \

Создать закладку: Option + Fn + F3

Автоформатирование кода: Cmd + Alt + L (если выделен фрагмент кода – то форматит только его, иначе – весь документ)

Метки в файле и в проекте (аналог Bookmarks - но проще и удобнее):
С помощью Ctrl + Shift + (0-9) можно оставлять метки с номерами.
Вернуться на метку можно с помощью Ctrl + (0-9) (номер метки).
Удалить метку – Ctrl + Shift + (номер метки)
Работает в контексте всего проекта.

Окружение выделения предзаданными штуками, типа тегов или if-else, в зависимости от контекста:
[CMD] + [ALT] + T (WIN: CTRL + ALT + T)

• Использование контекстного(нижнего) поля позволит вам изменять строку,
которая написана на другом языке в специальном редакторе.
Например, если у вас есть фрагмент HTML написанный в строке PHP,
вы можете ALT+ENTER перенестись в содержание данной строки и выбрать Edit HTML (редактирование) содержания.

Переход к месту последнего редактирования
Use keyboard shortcut ⇧⌘⌫. (Shift + Cmd + Delete)

Переход к строке №N: Cmd + L - и указать номер нужной строки

Переключение между открытыми файлами и Окнами Инструментов редактора: Ctrl + Tab
Удерживая Ctrl, используйте следующие варианты:
↑ (Up) and ↓ (Down) arrow keys, ⇥ (Tab) or ⇧⇥ (Shift + Tab) to go up and down the list in both panes.
Alt to toggle between the right and the left panes of the switcher pop-up window.
⌦ (Fn + Delete) or ⌫ (Delete) to close the editor tab where the selected file is opened, or hide a tool window, if it is shown.
